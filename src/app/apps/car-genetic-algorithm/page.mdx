  ## About

  This project demonstrates how a neural network can learn to drive a 2D car — purely through evolution, with no backpropagation or labelled training data. Cars that navigate the track better survive and pass their neural network "genes" to the next generation. Over successive generations, the population learns to drive.

  It is a practical, visual implementation of Neuroevolution — combining neural networks with genetic algorithms to solve a control problem.

  ## Challenge

  Train a neural network to drive a 2D car around a track without:

  1. Any pre-labelled training data.
  2. Backpropagation or gradient-based optimization.
  3. Human-defined rules for steering logic.

  The system must learn entirely through trial, error, and survival — mimicking natural selection.

<Section id="solution">

## Solution

<AppSolution href={appData.url}>

**2D Physics Simulation:** Built a Pygame-based 2D environment where each car has sensor rays that detect track boundaries. The sensor readings (distances to walls) feed directly into the neural network as inputs.

**Neural Network Architecture:** Each car runs a simple feedforward neural network — sensor inputs → hidden layer → steering and throttle outputs. Weights are encoded as a flat array (the "genome") that can be mutated and crossed over.

<ul role="list" className="mt-6 space-y-8">
  <li>
    <span>
      <strong className="font-semibold text-blue-100">Genetic Algorithm — Selection:</strong> After each generation, cars are ranked by a fitness function (distance traveled before crashing + time alive). The top N% are selected as parents for the next generation.
    </span>
  </li>
  <li>
    <span>
      <strong className="font-semibold text-blue-100">Crossover & Mutation:</strong> Parent genomes are combined (crossover at a random split point) and small random mutations are applied to weights — preventing the population from converging too early on a local optimum.
    </span>
  </li>
  <li>
    <span>
      <strong className="font-semibold text-blue-100">Real-Time Visualization:</strong> The simulation renders all cars simultaneously each generation — you can watch the population improve in real time. A live graph (Matplotlib) plots best fitness score per generation, showing the learning curve clearly.
    </span>
  </li>
  <li>
    <span>
      <strong className="font-semibold text-blue-100">Results:</strong> Within 15–20 generations on a moderately complex track, the best car consistently navigates the full course without crashing — emerging purely from random initial weights and selective pressure.
    </span>
  </li>
</ul>

</AppSolution>
